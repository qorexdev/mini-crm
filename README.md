A lightweight, browser-based CRM for managing leads and orders. No server, no database — everything runs locally in your browser using LocalStorage.

![Screenshot](https://via.placeholder.com/900x500/6366f1/ffffff?text=Mini+CRM+Preview)

## Features

- **Full CRUD** — create, read, update, delete leads
- **Status tracking** — New, In Progress, Done, Cancelled with color-coded badges
- **Search & filter** — instant search across all fields, filter by status
- **CSV import/export** — compatible with Excel and Google Sheets
- **Dark theme** — smooth toggle with system preference detection
- **Mobile-first** — responsive card layout on small screens, clickable phone numbers
- **Toast notifications** — visual feedback for every action
- **Demo data** — one-click sample data generation

## Getting Started

No Node.js or npm required.

```bash
git clone https://github.com/qorexdev/mini-crm.git
cd mini-crm
```

Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge).

> First load requires internet for Tailwind CSS CDN and Google Fonts. After caching, works offline.

## Tech Stack

| Technology | Usage |
