Легкая, современная и полностью автономная CRM-система, работающая прямо в браузере. Идеальное решение для фрилансеров и малого бизнеса: без сервера, без базы данных, без сложной установки.

![Mini CRM Preview](https://via.placeholder.com/1200x600/6366f1/ffffff?text=Mini+CRM+Preview)

## ✨ Возможности

- **🚀 Полная автономность**: Все данные хранятся локально в вашем браузере (LocalStorage). Ваши данные принадлежат только вам.
- **💎 Premium UI/UX**:
    - Современная цветовая палитра **Indigo**.
    - Профессиональная типографика **Inter** и **JetBrains Mono**.
    - Плавные анимации и микровзаимодействия (60 FPS).
- **🌓 Плавная темная тема**: Профессиональный ночной режим, который бережет ваши глаза.
- **📱 Mobile First**:
    - Полностью адаптивный интерфейс.
    - Удобное меню-"гамбургер" и карточки для мобильных устройств.
    - Кликабельные номера телефонов для мгновенного звонка.
- **⚡️ Быстродействие**:
    - Мгновенный поиск и фильтрация без перезагрузки.
    - Оптимизированный код на чистом JavaScript (Vanilla JS).
- **🛠 Функциональность**:
    - Управление заявками (CRUD): Создание, Чтение, Обновление, Удаление.
    - Статусы заявок с визуальной индикацией.
    - Импорт и Экспорт данных в формате CSV (совместимо с Excel).
    - Умные уведомления (Toast notifications).

## 🚀 Быстрый старт

Проект не требует установки Node.js, npm или настройки сервера.

1. **Скачайте код**:
   ```bash
   git clone https://github.com/qorexdev/mini-crm.git
   ```
2. **Запустите**:
   Просто откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).

> **Примечание**: Для первой загрузки требуется интернет-соединение для подключения стилей (Tailwind CSS) и шрифтов. После кэширования может работать офлайн (при соответствующей настройке PWA, в планах).
